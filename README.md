Airbnb Vancouver — Market Segmentation & BI Dashboard


Uncovering hidden listing segments and sentiment patterns across Vancouver's short-term rental market using unsupervised machine learning and NLP.




Project Overview

Vancouver's Airbnb market is saturated — but not all listings compete equally. This project segments the market into distinct clusters using K-Means clustering and PCA, then layers VADER sentiment analysis on top of guest reviews to understand why guests rate listings the way they do. The findings are visualised in an interactive Power BI / Tableau dashboard built for business decision-makers.

Business question: Which listing segments drive the highest satisfaction, and what pricing and location factors define them?


Key Findings

Sentiment analysis revealed that cleanliness and communication were the top drivers of positive reviews, while value-for-money was the leading source of negative sentiment
Dashboard enables filtering by neighbourhood, room type, price band, and sentiment score
